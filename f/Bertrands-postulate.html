📘 What is "n"?

n is simply the number you choose to start with.

For example, if you enter n = 10, the program will check all numbers between:

👉 10 and 20 (which is 2 × 10)

📘 The theorem

For every number n > 1, there is always at least one prime number between n and 2n.

🔍 What is a prime?

A prime number is a number greater than 1 that is only divisible by:

  • 1
  • itself

⚙️ What this shows

The program checks every number between n and 2n and highlights the primes in green.
